# 📚 NexoScienceSearch - Kotlin Android

**Versión nativa en Kotlin + Jetpack Compose**

Buscador de artículos científicos funcional.

## ⚡ Cómo generar el APK

1. Abre Android Studio
2. New Project > Empty Activity (Compose)
3. Copia el contenido de `MainActivity.kt` de este repositorio
4. Build > Build APK(s)

El APK estará listo en minutos.
