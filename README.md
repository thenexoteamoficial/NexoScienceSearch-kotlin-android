## 🚀 Descripción

NexoScienceSearch es una aplicación moderna para buscar y explorar artículos científicos de forma rápida y eficiente. Desarrollada completamente en Kotlin nativo con Jetpack Compose.

## ✨ Características actuales

- Búsqueda en tiempo real
- Filtrado por título y autor
- Interfaz moderna con Material 3
- Diseño limpio y profesional
- Datos de ejemplo realistas
- Arquitectura clara y mantenible

## 🔧 Cómo generar el APK

### Método recomendado (Android Studio)

1. Abre **Android Studio** (versión reciente)
2. Crea un nuevo proyecto: **Empty Activity** (selecciona **Jetpack Compose**)
3. Copia el contenido completo de `MainActivity.kt` de este repositorio
4. Ve a `Build > Build Bundle(s) / APK(s) > Build APK(s)`
5. El APK se generará en `app/build/outputs/apk/debug/`
